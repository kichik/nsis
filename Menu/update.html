Update

MakeNSISw allows you to check whether a new release is available. Choose NSIS Update in the Help menu to check for a newer version.

The releases have been tested and provide new features and fixes.
A nightly build of latest development files is also available on the website, at the bottom of the front page. Using the development files you can try the latest cutting-edge features and fixes. These files have not been tested so they should not be used in production environments.

See the Changelog for the latest changes in the development files.
Check the website for more development information.
